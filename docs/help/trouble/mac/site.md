# 一部のサイトが開けない"不具合"の解決法

## 発生する問題

稀に、一部のサイトが開けなかったり、通信ができない不具合が発生します。これはおそらく、Wi-Fi 周りの **"不具合"** だと推測しています。
以下の方法で対処できます。

## 解決法

1. Wi-Fi をオフにする
2. Wi-Fi をオンにする
3. 20~30 秒前後待つとインターネットにつながるので、開きたいサイトを開いたり、通信したいサイトをアクセスすると正常にアクセスすることができます。
